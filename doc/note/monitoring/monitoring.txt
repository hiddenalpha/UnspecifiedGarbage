
## QuickNDirty Overview (linux only)

  watch -n5 'true \
      && printf "loadavg: " && cat /proc/loadavg \
      && echo " --cpu----" && cat /proc/pressure/cpu \
      && echo " --mem----" && cat /proc/pressure/memory \
      && echo " --io----" && cat /proc/pressure/io \
      && echo " --disc----" && df -h | grep -E "^/" \
      && true'
